# ezAria2
一个ARIA2的windows GUI项目，完全基于.net，而非web ui加壳。

##开发进度

180311 现在允许添加一个HTTP/HTTPS的下载地址，能够正确将下载任务添加到ARIA2，并且实时更新下载进度和速度等信息。
180312 更新了一些显示元素

##预期功能
1. 使用GUI实现aria2的所有命令
2. 完成对aria2 Notice的解析
3. 通过其他开源库，完成对ed2k协议的支持（有生之年系列，求推荐些简单易懂的emule开源库）
